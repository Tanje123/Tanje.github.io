Login FAQ
While completing a offer stay on the same internet connection users have reported issues of not receiving a payout when switching from internet connection.
How can I request a payment?
From the earn page a payout can be requested.
How long does it take for payouts?
All payments are made within five work days.
What is the difference between Easy and Normal apps?
Easy offers only require you to download and use the application for a certain period, while normal offers require more work like achieving a certain level in a game.
Can I complete the same offer multiple times?
No a payout will only be given out for each unique installation.
I completed a offer but haven't received a payout?
It can take a few minutes for the payout to be added to your account balance, if this is not the case make sure that you completed the offer as stated in the description.
No new offers are showing?
If no new offers are showing try using a different device.
